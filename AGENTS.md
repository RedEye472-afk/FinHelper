# Public Datasets

При работе над функциями, связанными с финансами, экономикой, налогами, кредитами,
биржевыми данными или недвижимостью — используй `@public-datasets` из reference.

Там есть разделы:
- **Finance** — BIS, NASDAQ, FRED, SEC EDGAR, Google Finance, Quandl и др.
- **Economics** — макроэкономические данные, торговля, производительность
- **Government** — налоговые и бюджетные данные разных стран
- **RealEstate** — данные по недвижимости (если появится)

Ищи нужный датасет через README.rst в корне репозитория.
