# FinHelper — Справочные материалы для разработки

## Назначение

Данная директория содержит извлечённые тексты из учебных пособий по финансовой
математике и финансовой грамотности, а также составленные справочники из открытых
интернет-источников. Файлы предназначены для использования ИИ-ассистентом при
разработке приложения **FinHelper**.

## Покрытие тем

### Формулы и расчёты (Источник 1 — Копнова)
- Начисление простых и сложных процентов (декурсивный / антисипативный)
- Номинальные и эффективные ставки, непрерывное начисление
- Потоки платежей и аннуитеты (обычные, авансированные, бессрочные)
- Конверсия и консолидация платежей
- Погашение долга: дифференцированные и аннуитетные платежи
- Инвестиционные проекты: NPV, IRR, MIRR, срок окупаемости
- Облигации: бескупонные, купонные, серийные, аннуитетные
- Дюрация, выпуклость, иммунизация портфеля облигаций
- Акции: модели оценки доходности и стоимости
- Финансовые риски: количественные меры, критерии принятия решений
- Оптимизация портфеля (Марковиц, Шарп, Tobin)
- Модели CAPM и APT
- Эконометрическое моделирование (регрессия, мультиколлинеарность)
- Временные ряды: ARMA, ARIMA, GARCH, ARFIMA
- Производные: форварды, опционы, модель Блэка-Шоулза (Cox-Ross-Rubinstein)

### Личные финансы (Источник 2 — Горяев, Чумаченко)
- Личный финансовый план и бюджетирование
- Сбережения и накопления
- Банковские вклады и депозиты
- Кредиты, кредитные карты, ипотека
- Инвестиции: акции, облигации, ПИФы, ETF
- Страхование (жизнь, имущество, здоровье)
- Пенсионное планирование
- Налоги и налоговые вычеты
- Защита от мошенничества
- Финансовые продукты для бизнеса

### Продвинутые модели (Источник 3 — Ананьев, Гредасова)
- Портфельный анализ: Марковиц, Тобин, CAPM
- Безарбитражность и мартингальные меры
- Европейские и американские платёжные обязательства
- Биномиальная и гауссовская модели рынка
- Суперхеджирование
- Сходимость к цене Блэка-Шоулза
- Барьерные опционы, экзотические опционы
- Реализация в Matlab

### Быстрый справочник формул (Источник 4 — Составной)
- Проценты (простые, сложные, непрерывные, дисконтирование)
- Аннуитеты (обычный, авансированный, бессрочный)
- Кредиты и ипотека (аннуитетный платёж, дифференцированный, переплата)
- Инвестиции (NPV, IRR, MIRR, DPP, PI)
- Облигации (цена, YTM, дюрация, выпуклость)
- Акции (модель Гордона, P/E, P/B, дивидендная доходность)
- Инфляция и формула Фишера
- Портфельный анализ (CAPM, коэффициент Шарпа, бета)
- Ключевая ставка ЦБ РФ (справочная таблица)
- Excel-функции для финансовых расчётов

### Налоговые вычеты (Источник 5 — Составной)
- Имущественный вычет: покупка жилья (2 млн ₽) + проценты (3 млн ₽)
- Имущественный вычет: продажа имущества (минимальный срок владения)
- Социальный вычет: лечение (120K / без лимита), образование (120K / 50K), благотворительность
- Инвестиционный вычет: ИИС-1 (доходный) и ИИС-2 (пополнение, до 400K ₽)
- Стандартные вычеты: детские (1400/3000), ветераны (5000)
- Прогрессивная шкала НДФЛ 2025 (13% → 22%)
- Изменения 2025–2026: ставки, лимиты, правила
- Чек-лист документов для подачи
- Ключевая ставка ЦБ РФ: 21% → 18% → 14,5% → прогноз 10,5–11,5%

### Онлайн-ресурсы (Источник 6 — Составной)
- Финкульт.info (ЦБ РФ): официальный портал финансовой грамотности
- Калькуляторы: ипотека, сложные проценты, NPV/IRR, налоговые вычеты
- Образовательные курсы: ЦБ, OpenEdu, Coursera, Stepik
- Инвестиции: брокеры, аналитика (Smart-lab, RBC, MOEX, rusbonds)
- Защита от мошенничества: схемы, проверка организаций
- Налоги: ФНС, ЛК налогоплательщика, калькуляторы
- Пенсии: СФР, НПФ, софинансирование
- API для разработчиков: ЦБ РФ, MOEX

## Использование для FinHelper

### Как ИИ может использовать эти материалы:
1. **Расчётные модули** — формулы начисления процентов, аннуитетов, погашения кредитов, оценки облигаций (Источники 1, 4)
2. **Образовательный контент** — объяснения финансовых понятий для пользователей (Источники 2, 6)
3. **Советник по личным финансам** — бюджетирование, планирование, выбор финансовых продуктов (Источник 2)
4. **Инвестиционный анализ** — CAPM, оптимизация портфеля, оценка рисков (Источники 1, 3)
5. **Налоговое планирование** — расчёт вычетов, рекомендации по возврату НДФЛ (Источник 5)
6. **Проверка ответов** — верификация расчётов по формулам из учебников (Источники 1, 4)
7. **Ссылки на ресурсы** — рекомендации пользователям (калькуляторы, курсы, официальные сайты) (Источник 6)

### Рекомендации по интеграции:
- Источник 1 — основа для **модуля финансовых расчётов** (калькуляторы кредитов, депозитов, инвестиций)
- Источник 2 — основа для **модуля финансовой грамотности** (FAQ, обучающие материалы, советы)
- Источник 3 — основа для **продвинутого анализа** (портфельная теория, деривативы)
- Источник 4 — **быстрый справочник** для валидации расчётов и генерации формул
- Источник 5 — основа для **модуля налогового планирования** (вычеты, возврат НДФЛ)
- Источник 6 — **каталог внешних ресурсов** для рекомендаций пользователю

## Формат файлов

- **Источники 1–3** (извлечённые из PDF): содержат YAML frontmatter, описание и полный текст
  с разбивкой по страницам (`[Страница N]`)
- **Источники 4–6** (составленные): содержат YAML frontmatter, структурированные таблицы,
  формулы и чек-листы

Файлы могут использоваться как контекст для RAG-систем, для обучения моделей или как
справочник при разработке.
