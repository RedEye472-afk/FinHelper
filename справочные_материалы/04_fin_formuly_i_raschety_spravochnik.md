# Финансовые формулы и расчёты — Справочник FinHelper

> Компактный справочник ключевых финансовых формул для использования в расчётном ядре FinHelper.

---

## 1. Проценты

### 1.1 Простой процент (декурсивный)

```
S = P × (1 + i × n)
```

- `S` — наращенная сумма (future value)
- `P` — текущая стоимость (present value)
- `i` — процентная ставка (в долях, например 0.10 = 10%)
- `n` — срок (в годах)

### 1.2 Простой процент (антисипативный — авансовый)

```
P = S × (1 − d × n)
```

- `d` — учётная ставка (учётный дисконт)

### 1.3 Сложный процент

```
S = P × (1 + i)^n
```

### 1.4 Эффективная годовая ставка

```
i_eff = (1 + i_nom / m)^m − 1
```

- `i_nom` — номинальная годовая ставка
- `m` — число периодов начисления в году (12 для ежемесячного, 4 для квартального, 365 для ежедневного)

### 1.5 Непрерывное начисление

```
S = P × e^(δ × n)
```

- `δ` — сила роста (непрерывная ставка)
- `e` ≈ 2.71828

## 2. Аннуитеты

### 2.1 Аннуитетный платёж (классический)

```
A = P × [r × (1 + r)^n] / [(1 + r)^n − 1]
```

- `A` — размер аннуитетного платежа
- `P` — сумма кредита / начальная стоимость
- `r` — процентная ставка за период (месяц, квартал, год)
- `n` — количество периодов

**Пример:** Кредит 1 000 000 руб., 5 лет, 10% годовых, ежегодные платежи:
- `r = 0.10`, `n = 5`
- `A = 1 000 000 × [0.10 × 1.10^5] / [1.10^5 − 1] = 263 796.56 руб.`

**Важно:** При ежемесячных платежах:
- `r = годовая_ставка / 12`
- `n = лет × 12`

### 2.2 Приведённая стоимость обычного аннуитета

```
PV = A × [(1 − (1 + r)^(-n)) / r]
```

### 2.3 Будущая стоимость аннуитета

```
FV = A × [(1 + r)^n − 1) / r]
```

### 2.4 Аннуитет авансированный (пременумерандо)

```
PV_av = A × [(1 − (1 + r)^(-n)) / r] × (1 + r)
```

### 2.5 Бессрочный аннуитет (perpetuity)

```
PV = A / r
```

## 3. Кредиты и ипотека

### 3.1 Аннуитетный платёж (Excel: `ПЛТ`)

```
= ПЛТ(ставка; кол_периодов; сумма_кредита)
```

### 3.2 Платёж по процентам в периоде t

```
Int_t = Остаток_долга_t × r
```

### 3.3 Платёж по телу кредита в периоде t

```
Prin_t = A − Int_t
```

### 3.4 Остаток долга после платежа t

```
Balance_t = Balance_(t-1) − Prin_t
```

### 3.5 Дифференцированный платёж (постепенный)

```
Prin_t = P / n
Int_t = (P − Prin × (t − 1)) × r
Payment_t = Prin_t + Int_t
```

- Первые платежи самые крупные, последние — самые маленькие
- Общая переплата при дифференцированных < при аннуитетных

### 3.6 Реальная процентная ставка (с учётом комиссий)

```
Реальная ставка = IRR(поток платежей по кредиту)
```

Когда банк включает страховку, комиссии и другие «скрытые» платежи:
1. Построить денежный поток: +кредит на старте, −платежи каждый период
2. Рассчитать IRR этого потока
3. Это и есть реальная ставка

**Пример:** Кредит 1 000 000 руб., 10% годовых, + страховка 2%/год → реальная ставка ≈ 12.9%

## 4. Инвестиционные показатели

### 4.1 NPV — Чистая приведённая стоимость (Excel: `ЧПС`)

```
NPV = Σ [CF_t / (1 + r)^t]  для t = 0, 1, ..., n
```

- `CF_t` — денежный поток периода t
- `r` — ставка дисконтирования
- **NPV > 0** → проект выгоден
- **NPV < 0** → проект убыточен
- **NPV = 0** → безубыточный

**Важно в Excel:** Функция `ЧПС` дисконтирует **все** переданные значения, включая нулевой период. Поэтому начальные инвестиции нужно прибавлять отдельно:
```
= ЧПС(r; CF_1:CF_n) + CF_0
```

### 4.2 IRR — Внутренняя норма доходности (Excel: `ВСД`)

```
IRR = ставка r, при которой NPV = 0
```

- **IRR > ставки дисконтирования** → проект выгоден
- **IRR < ставки дисконтирования** → проект невыгоден

### 4.3 MIRR — Модифицированная внутренняя доходность (Excel: `МВСД`)

```
MIRR = (FV_положительных_CF / PV_отрицательных_CF)^[1/n] − 1
```

- Учитывает реинвестирование промежуточных потоков по ставке финансирования

### 4.4 DPP — Дисконтный срок окупаемости

```
DPP = минимальный t, при котором Σ [CF_k / (1+r)^k] ≥ 0  для k = 0..t
```

### 4.5 PI — Индекс доходности (рентабельности)

```
PI = Σ [CF_t / (1+r)^t] / I_0
```

- `I_0` — начальные инвестиции
- **PI > 1** → проект выгоден

## 5. Облигации

### 5.1 Бескупонная облигация (zero-coupon)

```
P = FV / (1 + r)^n
```

### 5.2 Купонная облигация с выплатой в конце срока

```
P = Σ [C / (1+r)^t] + FV / (1+r)^n  для t = 1..n
```

- `C` — купонный платеж
- `FV` — номинальная стоимость (face value)
- `r` — требуемая доходность (YTM)

### 5.3 Бессрочная облигация (perpetual bond)

```
P = C / r
```

### 5.4 Текущая доходность

```
Y_current = C / P_market × 100%
```

### 5.5 Доходность к погашению (YTM)

Решается уравнение относительно `r`:
```
P_market = Σ [C / (1+r)^t] + FV / (1+r)^n
```

### 5.6 Дюрация Маколея

```
D_Mac = Σ [t × PV(CF_t)] / P_bond
```

### 5.7 Модифицированная дюрация

```
D_mod = D_Mac / (1 + r)
```

### 5.8 Изменение цены облигации (приближённо)

```
ΔP / P ≈ −D_mod × Δr
```

## 6. Акции

### 6.1 Модель Гордона (дивидендный рост)

```
P = D_1 / (r − g)
```

- `D_1` — ожидаемый дивиденд в следующем году
- `r` — требуемая доходность
- `g` — постоянный темп роста дивидендов

### 6.2 Доходность акции

```
r = D_1 / P + g
```

### 6.3 Коэффициент P/E

```
P/E = Цена_акции / Прибыль_на_акцию (EPS)
```

## 7. Инфляция и налоги

### 7.1 Формула Фишера

```
r_real = (1 + r_nom) / (1 + i_inf) − 1
```

- `r_real` — реальная ставка
- `r_nom` — номинальная ставка
- `i_inf` — инфляция

**Приближение** (для малых значений):
```
r_real ≈ r_nom − i_inf
```

### 7.2 Налог на процентный доход (простой)

```
S_net = P × [1 + i × (1 − T) × n]
```

- `T` — ставка налога (в долях)

### 7.3 Налог на процентный доход (сложный)

```
S_net = P × [1 + i × (1 − T)]^n
```

## 8. Портфельный анализ

### 8.1 Ожидаемая доходность портфеля

```
r_p = Σ [w_i × r_i]
```

- `w_i` — доля i-го актива в портфеле
- `r_i` — ожидаемая доходность i-го актива

### 8.2 Риск портфеля (стандартное отклонение)

```
σ_p² = Σ Σ [w_i × w_j × σ_i × σ_j × ρ_ij]
```

- `σ_i` — стандартное отклонение i-го актива
- `ρ_ij` — коэффициент корреляции между активами i и j

### 8.3 Модель CAPM

```
r_i = r_f + β_i × (r_m − r_f)
```

- `r_f` — безрисковая ставка
- `r_m` — доходность рынка
- `β_i` — бета-коэффициент актива i

### 8.4 Модель Шарпа (рыночная модель)

```
r_i = α_i + β_i × r_m + ε_i
```

- `α_i` — альфа (сверхрыночная доходность)
- `ε_i` — случайная ошибка

### 8.5 Коэффициент Шарпа

```
Sharpe = (r_p − r_f) / σ_p
```
