# Финансовая грамотность: онлайн-ресурсы и инструменты

## Обзор

Данный документ содержит систематизированный каталог онлайн-ресурсов по финансовой
грамотности, полезных для пользователей FinHelper. Ресурсы сгруппированы по
темам и подходят для интеграции в модуль рекомендаций приложения.

---

## 1. Официальные государственные ресурсы

### 1.1. Финкульт (fincult.info)
**URL:** https://fincult.info
**Владелец:** Банк России (ЦБ РФ)
**Описание:** Главный портал по финансовой грамотности в России.

**Структура портала:**

| Раздел | Содержание |
|--------|------------|
| **Личные финансы** | Бюджетирование, сбережения, финансовое планирование |
| **Кредиты и заёмные средства** | Виды кредитов, как выбрать, как не переплатить |
| **Инвестиции** | Базовые знания: акции, облигации, ПИФы, ETF |
| **Страхование** | Виды страхования, как выбрать полис |
| **Пенсии** | Накопительная и страховая пенсия, софинансирование |
| **Налоги** | НДФЛ, вычеты, декларации |
| **Защита прав** | Как жаловаться на банки, страховые компании |
| **Цифровые финансы** | Мобильный банкинг, кибербезопасность |

**Для FinHelper:** Приложение может ссылаться на конкретные статьи fincult.info
при объяснении финансовых понятий. Портал является авторитетным источником,
утверждённым регулятором.

### 1.2. ЦБ РФ — Информация для граждан
**URL:** https://cbr.ru/queries/ (обратная связь, жалобы)
**URL:** https://cbr.ru/information/ (аналитика, статистика)

**Полезные разделы:**
- Ключевая ставка и макроэкономика
- Статистика по депозитам и кредитам
- Реестр финансовых организаций
- Информация о недобросовестных компаниях

### 1.3. ФНС России
**URL:** https://www.nalog.gov.ru
**URL ЛК налогоплательщика:** https://lkfl2.nalog.gov.ru

**Полезные разделы:**
- Калькуляторы налоговых вычетов
- Личный кабинет (подача 3-НДФЛ онлайн)
- Справочная информация по налогам
- Порядок получения вычетов

- Переплату по кредиту
- График платежей

### 2.2. Калькулятор сложных процентов

| Ресурс | Особенности |
|--------|-------------|
| Calc.ru — Сложные проценты | Ввод начальной суммы, ставки, периода, пополнений |
| Tinkoff (Т-Банк) — калькулятор доходности | Учёт пополнений, капитализации, налога |
| БКС — калькулятор вкладов | Сравнение вкладов разных банков |

**Формула для FinHelper:**
```
FV = PV × (1 + r/n)^(n×t) + PMT × [((1 + r/n)^(n×t) - 1) / (r/n)]

FV — будущая стоимость
PV — текущая стоимость (начальная сумма)
r — годовая процентная ставка (в долях)
n — количество капитализаций в год
t — количество лет
PMT — регулярное пополнение (аннуитетный постнумерандо)
```

## 5. Защита от мошенничества

### 5.1. Где проверять информацию

| Ресурс | URL | Что проверять |
|--------|-----|---------------|
| Банк России — реестр | cbr.ru | Легальность МФО, страховых компаний |
| ФНС — реестр | nalog.gov.ru | Статус ИП / юрлица |
| Федресурс | fedresurs.ru | Данные о банкротствах |
| Судебные приставы | fssp.gov.ru | Долги, исполнительные производства |

### 5.2. Основные схемы мошенничества (для обучения пользователей)

1. **Телефонное мошенничество** — «ваш аккаунт заблокирован», «ваш родственник в беде»
2. **Фишинг** — поддельные сайты банков, Госуслуг
3. **Инвестиционное мошенничество** — «гарантированная доходность 50%+», «пассивный доход»
4. **Крипто-скам** — фальшивые биржи, «наставники»
5. **Пирамиды** — финансовые пирамиды под видом «инвестиционных клубов»
6. **Кредитное мошенничество** — предложения «быстрого кредита без проверки»
7. **Социальная инженерия** — «служба безопасности банка» просит код

**Правило для FinHelper:** Приложение должно предупреждать пользователей о
недостоверных предложениях доходности и рекомендовать проверять все финансовые
организации через реестр ЦБ РФ.

## 9. Рекомендации для интеграции в FinHelper

### Модуль «Обучение»
- Ссылаться на fincult.info для подробных объяснений
- Использовать калькуляторы ЦБ и ФНС для расчётов
- Предоставлять ссылки на конкретные статьи при объяснении тем

### Модуль «Инвестиции»
- Показывать рейтинги с Investfunds и Московской биржи
- Ссылаться на обучающие материалы брокеров
- Выводить предупреждения о мошенничестве

### Модуль «Налоги»
- Ссылаться на калькуляторы ФНС
- Предоставлять инструкции по подаче 3-НДФЛ
- Рассчитывать потенциальный возврат по вычетам

### Модуль «Кредиты и ипотека»
- Интегрировать расчёты аннуитетных и дифференцированных платежей
- Ссылаться на актуальные средние ставки ЦБ
- Предоставлять ссылки на банковские калькуляторы

### Модуль «Безопасность»
- Показывать рекомендации по защите от мошенничества
- Ссылаться на реестр ЦБ для проверки организаций
