Возможности

Всё для управления финансами

FinHelper объединяет трекинг, планирование и аналитику в одном приватном приложении.

📊
Дашборд
Сводка доходов/расходов, баланс счетов, категоризация операций и проекция бюджета на одном экране.
💳
Операции
Доходы и расходы с идемпотентными ключами. Без дублей даже при сетевых сбоях. Категории с автоподстановкой.
🎯
Финансовые цели
Sinking Fund с учётом инфляции и доходности. What-if симуляции, проекции достижения цели, журнал пополнений.
🏦
Кредитный калькулятор
Аннуитет и дифференцированные платежи. ПСК по ЦБ (номинальная ставка). Досрочные погашения. График платежей.
💰
Депозитный калькулятор
Простые и сложные проценты, капитализация, пополнения. Эффективная ставка с учётом налогов (НДФЛ по ступеням).
📈
Инвестиции
NPV, IRR/XIRR, MIRR, PI. Анализ денежных потоков по методологии Копновой. Сравнение ипотеки vs аренды.
📋
Бюджеты
Месячные и годовые бюджеты с rollover остатков. Проекции на основе фактических трат и план-факт анализ.
🧮
Калькуляторы
Affordability (сколько можно взять), Mortgage vs Rent, налоги (НДФЛ, НПД, УСН), инфляционные поправки.
🔐
Zero-Trust
Все данные шифруются и хранятся локально. PII маскируется до логирования. Refresh rotation без хранения токенов.

Приватность

Ваши данные — только у вас

FinHelper спроектирован по принципу Privacy by Design в соответствии с 152-ФЗ.

🏠
Полностью локальный
Никакие финансовые данные не покидают ваше устройство. Сервер нужен только для аутентификации. Можно запустить на своём Docker-хостинге.
🛡️
PII-маскирование
Номера телефонов, email, паспортные данные, ФИО маскируются ещё до сохранения в БД. В логах — только user_hash.
🔑
Refresh Rotation
Access token (15 мин) живёт только в памяти. Refresh token (30 дней) — httpOnly Secure cookie. В БД — только SHA-256 хеш.
📜
Open Source
Исходный код полностью открыт. Вы можете проверить, что приложение действительно не отправляет данные вовне.
