# Excel Processing

This skill helps with creating, editing, and processing spreadsheet files.

## Capabilities

- Create new spreadsheets
- Read and write data to cells
- Apply formulas and functions
- Create charts and graphs
- Format cells and ranges
- Process CSV files
- Data analysis and transformations

## Supported Formats

- Excel (.xlsx, .xls)
- CSV (.csv)
- TSV (.tsv)

## Usage

Use this skill when you need to:
- Process financial data in spreadsheets
- Create budget templates
- Analyze numerical data
- Generate reports from Excel files
- Convert between spreadsheet formats
