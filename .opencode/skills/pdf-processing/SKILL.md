# PDF Processing

This skill helps with creating and processing PDF files.

## Capabilities

- Create PDF documents from text or HTML
- Extract text content from PDFs
- Merge multiple PDFs into one
- Split PDF into separate pages
- Add watermarks or stamps
- Convert other formats to PDF
- Extract tables and data from PDFs

## Supported Operations

- PDF creation
- Text extraction
- PDF manipulation (merge, split)
- Format conversion
- Metadata extraction

## Usage

Use this skill when you need to:
- Generate financial reports as PDF
- Extract data from PDF statements
- Combine multiple documents into one PDF
- Process PDF invoices or receipts
- Create PDF summaries of financial data
