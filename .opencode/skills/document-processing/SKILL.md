# Document Processing

This skill helps with creating, editing, and processing text documents.

## Capabilities

- Create new documents in various formats (.txt, .md, .docx)
- Edit and modify existing documents
- Extract content from documents
- Format and structure document content
- Merge or split documents
- Search and replace text across documents

## Supported Formats

- Plain text (.txt)
- Markdown (.md)
- Word documents (.docx) - requires additional libraries

## Usage

Use this skill when you need to:
- Create financial reports or summaries
- Process text data from files
- Generate documentation
- Manipulate document content programmatically
