# Presentation Creation

This skill helps with creating and editing presentation files.

## Capabilities

- Create new presentations from scratch
- Add and format slides
- Insert text, images, charts, and shapes
- Apply themes and templates
- Export presentations to different formats
- Convert between presentation formats

## Supported Formats

- PowerPoint (.pptx, .ppt)
- Keynote (.key) - limited support
- PDF export

## Usage

Use this skill when you need to:
- Create financial presentation decks
- Build data visualization slides
- Generate automated reports in presentation format
- Design professional business presentations
